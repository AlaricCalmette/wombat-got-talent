## wombat got talent

### Dev dependencies
* [Elixir](https://elixir-lang.org/install.html)
* [PostgreSQL](http://postgresapp.com/)
* [Phoenix](http://www.phoenixframework.org/docs/installation)

### Running the app
```bash
cd wombat_got_talent;
mix phoenix.server;
```

go to ```localhost:4000```
